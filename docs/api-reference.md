# Referencia publica de API e webhooks

A pagina `/docs/api` documenta os endpoints tecnicos do GitHub DevLog AI.

## Endpoints cobertos

- `POST /webhooks/github/{workspaceUuid}`
- `POST /webhooks/github-app`
- `POST /webhooks/mercado-pago`

## Conteudos

- Headers esperados.
- Validacao de assinatura.
- Respostas comuns.
- Erros comuns.
- Boas praticas de seguranca.

## Objetivo

- Ajudar devs a integrar sem depender do painel.
- Apoiar revisao do GitHub App.
- Reduzir chamados de suporte.
- Fortalecer o readiness de go-live.
