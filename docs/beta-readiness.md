# Prontidao beta/local

A pagina `/admin/beta-readiness` separa o que pode ser fechado localmente do que depende de producao.

## Prontidao local

- Cadastro/login.
- Workspace privado.
- Permissoes por papel.
- Webhook GitHub manual.
- Webhook GitHub App preparado.
- Mercado Pago preparado.
- Planos no banco.
- Suporte com SLA.
- Docs de usuario.
- Status publico.
- Auditoria.
- Roadmap admin.

## Dependencias externas

- Dominio definitivo com HTTPS.
- GitHub App oficial configurado.
- Mercado Pago em producao.
- E-mail transacional real.

## Uso

Antes de chamar alguem para testar, abra `/admin/beta-readiness` e confirme que a prontidao local esta alta. Pendencias externas devem ficar claras e separadas.
