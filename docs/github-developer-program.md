# GitHub Developer Program

Este documento organiza o pacote de submissao do GitHub DevLog AI para o GitHub Developer Program.

## Proposta curta

O GitHub DevLog AI e um inbox privado e auditavel para webhooks do GitHub. Ele ajuda devs e times a validar entregas, debugar eventos, preservar historico e demonstrar fluxos de integracao sem depender de logs locais ou payloads perdidos.

## Dor que resolve

- Webhooks chegam, mas o time nao sabe se a assinatura validou.
- Payloads ficam espalhados em logs locais.
- Demos dependem de terminal, tunnel e prints.
- Times precisam separar eventos por workspace e controlar acesso.
- Falhas de integracao exigem historico, contexto e auditoria.

## Publico

- Desenvolvedores que integram apps com GitHub.
- Times de produto que dependem de eventos GitHub.
- Agencias e squads que demonstram integracoes para clientes.
- Criadores de ferramentas que precisam testar webhooks com seguranca.

## Evidencias necessarias

- URL publica do produto.
- Demo funcional com repositorio GitHub real.
- Documentacao de configuracao.
- Explicacao das permissoes do GitHub App.
- Politica de privacidade.
- Termos de uso.
- Canal de suporte.
- Status publico.
