# Kit operacional de go-live

O painel `/admin/go-live-kit` organiza as ações finais que dependem do ambiente real.

## Blocos

- domínio e infraestrutura;
- GitHub App oficial;
- Mercado Pago produção;
- screenshots finais de submissão.

## Objetivo

Manter o roadmap local fechado e deixar claro que o restante depende de super admin, credenciais oficiais e publicação em domínio definitivo.

## Uso recomendado

Antes de submeter ao GitHub Developer Program, execute o checklist do painel, capture evidências finais e compare com `/admin/go-live`, `/admin/launch-overview` e `/admin/release-evidence`.
