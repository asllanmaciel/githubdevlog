# Prontidão geral de lançamento

O painel `/admin/launch-overview` consolida a visão geral do lançamento.

## Fórmula atual

O percentual geral é a média simples de quatro áreas:

- Beta/local.
- Go-live.
- GitHub Developer Program.
- Roadmap.

## Percentual atual

No momento da criação deste painel:

- Beta/local: 100%.
- GitHub Program: 100%.
- Roadmap: 81%.
- Go-live: 71%.
- Geral: 88%.

## Interpretação

O produto está maduro para beta e demonstrações, mas ainda não está liberado para live público porque os bloqueadores restantes dependem de ambiente externo.

## Bloqueadores externos principais

- Domínio oficial com HTTPS.
- GitHub App oficial.
- Mercado Pago em produção.
- E-mail transacional real.
