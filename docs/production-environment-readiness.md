# Ambiente de produção

O painel `/admin/production-env` mostra a prontidão técnica das variáveis críticas de produção.

## O que acompanha

- Domínio oficial e `APP_URL`.
- Fila assíncrona.
- E-mail transacional.
- GitHub App oficial.
- Mercado Pago produção.
- Contato público de suporte.

## Segurança

Valores longos ou sensíveis são mascarados na interface para evitar exposição acidental.
