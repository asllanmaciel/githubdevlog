# Changelog publico

O GitHub DevLog AI possui uma pagina publica de changelog em `/changelog`.

## Objetivo

- Mostrar evolucao do produto.
- Reforcar confianca para devs.
- Apoiar beta, demos e submissao ao GitHub Developer Program.
- Transformar entregas concluidas do roadmap em comunicacao publica.

## Origem dos dados

As entradas vem de `roadmap_items` com `status = done`.

## Cuidados antes do live

- Evitar descricoes internas demais.
- Remover detalhes sensiveis de infraestrutura.
- Revisar linguagem comercial.
- Manter as entregas mais recentes e relevantes.
