# Contato publico

O GitHub DevLog AI possui uma pagina publica de contato em `/contact`.

## Objetivo

- Dar um canal publico para usuarios antes do cadastro.
- Apoiar revisao do GitHub App.
- Facilitar contato de seguranca.
- Reduzir friccao comercial em demos e beta.

## Configuracao

O email exibido vem de:

- `DEVLOG_SUPPORT_EMAIL`

## Antes do live

- Trocar o placeholder pelo email oficial.
- Configurar caixa de entrada monitorada.
- Definir SLA publico.
- Separar contato de seguranca se necessario.
