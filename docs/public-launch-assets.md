# Ativos publicos de launch

O GitHub DevLog AI agora possui ativos basicos para operar como produto publico.

## URLs

- `/sitemap.xml`
- `/robots.txt`
- `/pricing`
- `/security`
- `/privacy`
- `/terms`

## Objetivo

- Facilitar indexacao e descoberta.
- Bloquear areas privadas em crawlers.
- Reforcar confianca publica.
- Apoiar revisao do GitHub App.
- Organizar a navegacao publica antes do dominio oficial.

## Revisar antes do dominio final

- `APP_URL` em producao.
- URLs canonicas do sitemap.
- Politicas publicas com contato real.
- Links do rodape.
- Descricao SEO do layout publico.
