# Paginas publicas de confianca

Antes do live, o GitHub DevLog AI precisa expor paginas publicas que expliquem privacidade, termos e seguranca.

## URLs

- `/privacy`
- `/terms`
- `/security`

## Objetivo

Essas paginas servem para:

- apoiar revisao do GitHub App;
- dar seguranca para usuarios tecnicos;
- explicar como payloads de webhook sao tratados;
- preparar a operacao para pagamentos reais;
- reduzir friccao comercial em demos e beta.

## Pontos a revisar antes do live

- Email oficial de contato.
- Razao social ou titular responsavel.
- Politica de retencao.
- Procedimento de exclusao/exportacao.
- Termos comerciais definitivos.
- Linguagem juridica final revisada.
