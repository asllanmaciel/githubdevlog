# Guia de usuário do GitHub DevLog AI

Este guia resume a experiência que um dev deve ter ao usar o produto.

## O que o usuário faz

1. Cria uma conta ou aceita convite de workspace.
2. Copia Payload URL e Secret.
3. Configura webhook manual no GitHub ou instala GitHub App.
4. Envia ping/push.
5. Acompanha evento no painel.
6. Cria nota ou tarefa quando precisa investigar.
7. Controla uso, plano, retenção e suporte.

## Ações por papel

- Owner/admin: equipe, billing, secrets e GitHub App.
- Developer: eventos, testes, notas e tarefas.
- Viewer: leitura e suporte.

## Mensagem do produto

O DevLog AI é um inbox privado e auditável para webhooks do GitHub. Ele reduz ruído, valida assinatura e transforma payload bruto em diagnóstico compartilhável.
